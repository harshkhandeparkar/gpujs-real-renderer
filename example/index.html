• # Progressive Line Graph

    Adds a new value to graph in real time and interpolates (makes a line) joining the value. Can be used to monitor real time data.
    Accommodates more data by moving behind (continously or on overflow) or squeezing the graph.

    Class: RealLineGraph



  • # Drawing Board

    A general purpose whiteboard/blackboard/[whichever color you want]board.

    Class: RealDrawBoard




  • # Random Symmetric Fourier Series

    Plot the sum of a randomly generated (symmetric) fourier series (10 numbers) using the RealComplexSpace class which can track, plot and interpolate (make a line between) complex numbers. This class also exports a util class named `Complex` for handling complex numbers.

    Class: RealComplexSpace
